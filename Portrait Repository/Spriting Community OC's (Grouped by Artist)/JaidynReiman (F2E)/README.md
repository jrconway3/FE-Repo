A characters are owned by me, JaidynReiman. Any assets can be used and edited freely, please simply credit me.

The initial portraits were generated using AI and redrawn and edited from there. I took inspiration from existing portraits to fine-tune the details and make it mesh better with the FE style (however, none of these portraits were spliced from pre-existing portraits except my own).

I am purposely providing alternate variations of portraits for the purpose of expanding options for people who want to use them. Most of my characters have elven style ears, but I have provided normal "human" ear variants.

The subdirectory "Parts" includes each individual asset's separate layer. You can use this to more easily create your own splices using these assets. It might even work with Vale the Violet Mote's character builder, but you'd have to repalette the assets to that style.
